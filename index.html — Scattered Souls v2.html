FAQ
What is Scattered Souls?+
Scattered Souls is a collection of 10,000 fully on-chain generative particle portraits on Ethereum. Each token is a unique silhouette built from hundreds of colored particles, generated entirely from smart contract code with no external dependencies.
What does "fully on-chain" mean?+
Every pixel, every color, every trait — all generated and stored directly on the Ethereum blockchain. No IPFS, no servers, no cloud storage. As long as Ethereum exists, your soul exists.
How does minting work?+
Each wallet gets 2 free mints via the freemint() function. Additional tokens cost 0.0001 ETH each, up to 20 per transaction. Total supply is capped at 10,000.
Are all souls truly unique?+
Yes. Each soul's traits — head shape, hair, palette, skin tone, aura and more — are derived deterministically from the token ID using on-chain randomness. No two souls share the exact same combination.
What are the royalties?+
2% royalty on secondary sales, enforced via the ERC-2981 standard. This helps sustain the project long-term.
Where can I trade?+
Scattered Souls is available on OpenSea and any marketplace that supports ERC-721 tokens on Ethereum mainnet.
